# Infinisort
 Uses Javascript to visualize sorting algorithms on an HTML5 canvas.
 By default, the program shuffles the array, selects a random sorting algorithm, sorts, and then repeats.
 
# Available algorithms:
 1. Selection Sort
 2. Insertion Sort
 3. Bubble Sort
 4. Cocktail Shaker Sort
 5. Quicksort
    - Maximum Pivot
    - Minimum Pivot
    - Median Pivot
    - Random Pivot
 6. Heapsort
    - Max Heap
    - Min Heap
 7. Odd-Even Sort
 8. Gnome Sort
 9. Comb Sort
 10. Circle Sort
 11. Cycle Sort
 12. Radix LSD Sort
    - Works with any base
    - Presets are 2, 4, 8, 10, 16, and 32
